# Event Module

- É um mecanismo para:
    - Disparar eventos
    - Ouvir eventos
    - Fazer alguma ação quando ouvir o evento
    - É a base para muitos outros módulos como: http, stream, file system, etc...

- Aprendizado:
    - Utilizar os eventos
    - Disparar eventos
    - Ouvir eventos
    - Executar ações para determinar eventos
    - Entender como ele é a base para os outros módulos
