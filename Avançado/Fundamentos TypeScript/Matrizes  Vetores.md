# Matrizes

- Existem duas maneiras de tipar arrays no TypeScript, confira os exemplos:

let numbers: number[];
numbers = [1,2,3,4,5,6];

let numbers: Array<number>
numbers = [1,2,3,4,5,6];

OU

let users: Array<string>;
users = ['Rodrigo', 'João'];

let users: string[];
users = ['Rodrigo', 'João'];
