FlexBox

- Nos permite posicionar os elementos dentro da caixa.
- Constrole em uma dimensão (horizontal ou vertical)
- Alinhamento, direcionamento, ordenar e tamanhos

div.parent {
	display: flex;
}

* Flex-direction
Qual a direção do flex: 
- row | column (horizontal ou vertical)


* Alinhamento
- justify-content
- align-items
